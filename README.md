AI contest hosted by Gangwon Fire Department

# 데이터 EDA(1)
## 데이터셋 기본 정보 파악
- 유동인구는 성별/연령별 유동정도에 따른 float 수치
  - 남성/여성 및 0세부터 5세 단위로 70세까지 존재
- 31개 사고유형
  - 낙상사고, 보행자사고, 자동차사고, 오토바이사고 등

## 가설 설정 및 검증
### 가설-1
- 유동인구가 없는 격자는 응급출동이 일어나지 않는다.
### 가설-2
- 응급출동이 발생했으면 1, 발생하지 않았으면 0으로 파생변수를 만들 수 있다.
